Full-page · l'écran de chargement principal

À utiliser pour first paint, route change majeur, app launch

Splash & route loader

Le loader signature JIM.

Le logo se reconstruit à chaque cycle — d'abord les jambages J·I·M, puis les deux points qui retombent à leur place. Évoque la promesse produit : le cabinet se remet en marche.

  • Cycle de 2,4 s — non agressif sur connexions lentes
  • Couleur corail unique sur fond marron foncé
  • Barre de progression auxiliaire indéterminée
  • Respect de prefers-reduced-motion (animation simplifiée)
